Important FAQ About DOM

What is the Document Object Model (DOM)?

Document Object Model (DOM) is a Programming interface for Web document. It defines the logical structure of documents and the way a document is accessed and manipulated.

How do you select an element from the DOM?

There are a few different methods for selecting an element in the HTML document. Most used are :

  1. 1.getElementById()
  2. 2.getElementsByClassName()
  3. 3.querySelector()
  4. 4.querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation leverages event propagation to manage events efficiently. Instead of adding event listeners to multiple child elements, you add a single event listener to a parent element. This listener takes advantage of event bubbling to handle events for its child elements.

How do you manipulate an element's attributes and styles using the DOM?

DOM manipulation allows developers to interact with and modify the structure, style, and content of web pages.To manipulate the DOM, we need to access its elements. This is commonly done using the document object, which represents the entire HTML document.DOM manipulation also extends to modifying the styles of elements, allowing us to create visually appealing and dynamic web pages.We can use the style property of an element to change its appearance. 

// Accessing an image element
    const myImage = document.getElementById('myImage');
    
    // Changing the source attribute of the image
    myImage.src = 'new-image.jpg';
Here, we access an image element with the ID myImage and change its src attribute to new-image.jpg, dynamically updating the displayed image.
